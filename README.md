## 🛠️ Como rodar o projeto?

1. Faça clone do repositório

```bash
git clone https://github.com/estartando-devs/site.git
```

2. Navegue até o diretório

```bash
cd site
```

3. Instale as dependencias

```bash
yarn install
```

4. Rode o app

```bash
yarn dev
```
